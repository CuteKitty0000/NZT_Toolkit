## 📱 Overview

**NZT Toolkit** combines essential cybersecurity and developer utilities into a single, clean, offline-friendly Android application.

Whether you're testing APIs, generating hashes, verifying checksums, or creating secure passwords, NZT Toolkit provides quick access to the tools you use every day.

---

## ✨ Features

- 🔒 Hash Generator
- 🔑 Password Generator
- 🔐 Password Strength Checker
- 📄 Base64 Encode & Decode
- 🪪 JWT Decoder
- 🌐 URL Encode & Decode
- ⏰ Unix Timestamp Converter
- ✅ Checksum Generator
- 📱 QR Code Generator
